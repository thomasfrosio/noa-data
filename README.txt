Test data for noa
This is meant to be an artefact repository.

Usage:
    - git clone --depth=1 https://github.com/thomasfrosio/noa-data.git && cd noa-data
    - Execute GenerateData.sh
    - Set the environmental variable NOA_DATA_PATH to this repository's path.
      This variable is used by noa_test to locate the test data.
